1. An IoT Based Voice Controlled Home Automation System

This mini project presents an IoT-based Voice Controlled Home Automation System that utilizes a WiFi module and the Sinric Pro server platform to enable seamless voice control of smart home devices. The objective of this project is to design and implement a user-friendly system that allows homeowners to control and monitor their smart home devices using voice commands. The system architecture comprises two main components: the WiFi module and the Sinric Pro server. The WiFi module establishes a wireless connection between the user's voice input device and the home automation devices. It acts as a bridge, enabling bidirectional communication and data exchange. The Sinric Pro server, a cloud-based platform, serves as the intermediary between the voice input device and the home automation devices, facilitating the integration and control of multiple devices.



Tech Stack : ESP8266, 4 channel Relay, Sinric Pro server
